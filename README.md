## Featured Projects

### [MT-SICS Emulator](https://github.com/mdcurt/mt-sics-emulator)
[![PyPI version](https://img.shields.io/pypi/v/mt-sics-emulator)](https://pypi.org/project/mt-sics-emulator/)
[![CI](https://github.com/mdcurt/mt-sics-emulator/actions/workflows/ci.yml/badge.svg)](https://github.com/mdcurt/mt-sics-emulator/actions/workflows/ci.yml)

Open-source server-side emulator for the MT-SICS scale protocol.

Built to help developers test integrations with industrial scales, warehouse systems, label printers, and automation scripts without needing physical hardware.

**Tech:** Python, TCP sockets, serial/RS-232, protocol design

### [Carrier Scorecard](https://github.com/mdcurt/alan-wire-carrier-scorecard)
A freight carrier performance dashboard for analyzing shipment costs, service reliability, and carrier performance.

Built as an operational analytics tool for comparing real-world logistics data.

**Tech:** Angular, TypeScript, data visualization, dashboard design
